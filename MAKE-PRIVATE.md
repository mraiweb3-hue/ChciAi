# Jak změnit GitHub repo na PRIVATE

## Krok 1:
1. Otevři: https://github.com/mraiweb3-hue/ChciAi
2. Klikni nahoře vpravo: **Settings**

## Krok 2:
3. Sjeď úplně dolů na "Danger Zone"
4. Najdi: **Change repository visibility**
5. Klikni: **Change visibility**

## Krok 3:
6. Vyber: **Make private**
7. Napiš název repa pro potvrzení: `ChciAi`
8. Klikni: **I understand, change repository visibility**

✅ HOTOVO - Repo je private!

## Ověření přístupů:
9. Settings → Collaborators and teams
10. Měl bys vidět jen sebe (+ případně Clawdbot app)
11. Pokud tam je něco dalšího → Remove

## Zkontroluj GitHub Apps:
12. Tvůj GitHub profil → Settings → Applications
13. Installed GitHub Apps → Clawdbot/Emergent
14. Ujisti se že má přístup JEN k ChciAi repo (ne ke všem)
